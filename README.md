# Go

### Requirements
- CMake v3.10.0
- Make
- SDL 2.0
- C++20 compiler

## Linux

Installing CMake
```
sudo apt install cmake
```
Installing Make
```
sudo apt install make
```
Installing SDL 2.0
```
sudo apt install libsdl2-dev
```
Installing a C++20 compiler (GCC version 8 and later)
```
sudo apt install g++
```

## Compiling
```
mkdir build
cd build
cmake ..
cmake --build .
```
